# Changelog

## v1.0.0

Initial public release of APG-Core.

Includes:

- squared Euclidean APG weights,
- Shannon entropy,
- normalized entropy,
- Renyi entropy,
- APG concentration,
- power-sum defect,
- Shannon--Renyi bounds,
- concentration lower bound,
- local closure defect,
- first-order entropy approximation,
- command-line interface,
- examples and tests.
